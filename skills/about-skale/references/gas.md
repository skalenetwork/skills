# Gas Models

## sFUEL (Zero Gas)

- Valueless gas token for EVM compatibility
- Users pay ~0; validators stake SKL to cover
- Prevents spam
- Europa, Calypso, Nebula

## Compute Credits

- Prepaid credits for compute on SKALE Base
- Can be purchased
- No PoW option

## Proof-of-Work

- Solve puzzle to send without sFUEL
- Disabled by default on credit chains

## Gas Chains

- Real token as gas (ETH, SKL, WBTC, ERC-20)
- Like traditional L2 setup

## Comparison

| Model | User Pays | Chains |
|-------|-----------|--------|
| sFUEL | ~0 | Europa, Calypso, Nebula |
| Compute Credits | Credits | SKALE Base |
| PoW | ~0 | Optional |
| Gas Chain | Token | Configurable |

## How to Get Gas on the Different Chains

For SKALE Base Chains:
- SKALE Base Mainnet: The gas or CRREDITS need to be purchased. They can be aquired on the SKALE Portal here: https://base.skalenodes.com/credits
- SKALE Base Testnet: The gas or CRREDITS can be aquired on the faucet here: https://base-sepolia-faucet.skale.space/;

For SKALE Ethereum-connected chains:
- For Europa Hub, Calypso Hub and Nebula Hub mainnets the gas or sFUEL are provided by the chain owner once the integration process is completed
- For Europa Hub, Calypso Hub and Nebula Hub testnets the gas or sFUEL can be acquired here: https://www.sfuelstation.com/;
